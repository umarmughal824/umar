# Umar Package

A simple Python package for greeting and adding numbers.

## Installation

You can install the `umar` package from PyPi:

```bash
pip install umar
```

```
from umar.greet import greet

greet("Umar Asghar")
```
This will print
```
Umar Asghar
```

Another function
```
from umar.add import add

add(2, 2)
```
This should print
```
4
```
